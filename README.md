# datastructuresandalgorithms

# Usage
```bash
# from project root
cd /pathto/datastructuresandalgorithms/python

# create venv
python3 -m venv .venv

# activate venv
source .venv/bin/activate

# upgrade pip and install requirements
python -m pip install --upgrade pip
pip install -r requirements.txt

# run all tests (or use `python -m pytest` if `pytest` command not found)
pytest -q
# or
python -m pytest -q
```
